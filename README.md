# PlayFab <-> Discord  Bot
A simple Discord bot to check PlayFab statistics and grant roles on Discord.

## Setup
1. **Install:** `npm install`
2. **Configure:** Set environment variables in `.env`:
- `DISCORD_BOT_TOKEN`
- `PLAYFAB_API_KEY`
- `PLAYFAB_TITLE_ID`
- `CHANNEL_ID`
3. **Run:** `npm start`
